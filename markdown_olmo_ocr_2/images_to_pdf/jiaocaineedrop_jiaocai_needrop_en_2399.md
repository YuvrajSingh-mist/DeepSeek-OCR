    <td>(1) 等式两边都加上(或都减去)同一个数或同一个整式，所得结果仍是等式。如果 \(a = b\)，那么 \(a + c = b + c, a - c = b - c\);<br>(2) 等式两边都乘以(或都除以)同一个数(除数不能为0)，所得结果仍是等式。如果 \(a = b\)，那么 \(ac = bc, \frac{a}{c} = \frac{b}{c} (c \neq 0)\)</td>

  </tr>
