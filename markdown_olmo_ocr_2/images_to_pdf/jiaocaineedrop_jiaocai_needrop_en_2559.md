【解析】
正四面体ABCD中，\( AB = \sqrt{2} \)，图中点O为外接球的球心，半径为\( R = OA = OB \)，\( O_1 \)为\( \triangle BCD \)的外心，
所以\( O_1B = \frac{1}{2} \times \frac{\sqrt{2}}{\sqrt{3}} = \frac{\sqrt{2}}{\sqrt{3}} = \frac{\sqrt{6}}{3} \)，由于\( O_1B^2 + OO_1^2 = OB^2 \)，
又因为\( O_1A = \sqrt{(\sqrt{2})^2 - \left( \frac{\sqrt{6}}{3} \right)^2} = \frac{2\sqrt{3}}{3} \)，所以\( \left( \frac{\sqrt{6}}{3} \right)^2 + \left( \frac{2\sqrt{3}}{3} - R \right)^2 = R^2 \)，解得\( R = \frac{\sqrt{3}}{2} \)，
因此外接球的表面积为\( 4\pi \times \left( \frac{\sqrt{3}}{2} \right)^2 = 3\pi \)，故A正确；
由于\( BE = \frac{\sqrt{6}}{2} \)，\( BO_1 = \frac{\sqrt{6}}{3} \)，\( AO_1 = \frac{2\sqrt{3}}{3} \)，且\( AB \)与平面BCD所成的角为\( \angle ABO_1 \)，
因此\( \sin \angle ABO_1 = \frac{AO_1}{AB} = \frac{\frac{2\sqrt{3}}{3}}{\sqrt{2}} = \frac{\sqrt{6}}{3} \)，故B错误；
因为\( AE \perp CD \)于E，所以\( AM_{min} = AE; BE \perp CD \)于E，所以\( BM_{min} = BE \)；
因此当M与E点重合时，\( AM + BM \)最小，最小值为\( 2 \times \frac{\sqrt{6}}{2} = \sqrt{6} \)，故C正确；
在平面ABC中过点T作PT\( \perp \)AB交AC于P，在平面ADC中过点T作RT\( \perp \)AB交AD于R，连接PR，
又因为\( RT \cap PT = T \)，所以\( AB \perp \)平面TPR，因此平面TPR即为所求，
\( TP = TR = \frac{\sqrt{6}}{3} \)，\( AD = PR = \frac{2\sqrt{2}}{3} \)，
则\( \triangle TPR \)的周长为\( \frac{\sqrt{6}}{3} + \frac{\sqrt{6}}{3} + \frac{2\sqrt{2}}{3} = \frac{2\sqrt{6} + 2\sqrt{2}}{3} \)，
同理在平面ABC中过点N作NQ\( \perp \)AB交BC于Q，在平面ABD中过点N作NS\( \perp \)AB交BD于S，连接QS，可得平面NQS，而平面NQS即为所求，
\( NQ = NS = \frac{\sqrt{6}}{3} \)，\( BQ = QS = AP = \frac{2\sqrt{2}}{3} \)，
则\( \triangle NQS \)的周长为\( \frac{\sqrt{6}}{3} + \frac{\sqrt{6}}{3} + \frac{2\sqrt{2}}{3} = \frac{2\sqrt{6} + 2\sqrt{2}}{3} \)，故D正确。
故选：ACD。
