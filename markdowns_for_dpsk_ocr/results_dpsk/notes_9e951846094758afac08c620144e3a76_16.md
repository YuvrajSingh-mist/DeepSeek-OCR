7. 如图, 梯形 \(ABCD\) 中, \(AB \parallel CD\). 若 \(AB = 1\),
\(CD = 2\), \(AC = 6\). \(S_{\triangle AOB} = 1\). \(RI \parallel AO = \_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\)
\(S_{\text{梯形} ABCD} = \_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_

<|ref|>text<|/ref|><|det|>[[162, 440, 465, 476]]<|/det|>
解: ∵ \(AB \parallel CD\) 

<|ref|>equation<|/ref|><|det|>[[245, 479, 567, 510]]<|/det|>
\[\therefore \triangle AOB \sim \triangle COD\]

<|ref|>equation<|/ref|><|det|>[[238, 509, 864, 565]]<|/det|>
\[\text{即} \frac{AO}{CO} = \frac{AB}{CD} \quad \because AB = 1, CD = 2, AC = 6\]

<|ref|>equation<|/ref|><|det|>[[260, 569, 660, 621]]<|/det|>
\[\frac{AO}{CO} = \frac{1}{2} \Rightarrow CO = 2AO\]

<|ref|>equation<|/ref|><|det|>[[192, 627, 890, 666]]<|/det|>
\[\therefore AC = AO + CO = 3AO \Rightarrow 3AO = 6, AO = 2\]

<|ref|>equation<|/ref|><|det|>[[150, 660, 863, 730]]<|/det|>
\[\therefore \frac{S_{\triangle AOB}}{S_{\triangle COD}} = \left(\frac{AB}{CD}\right)^2 = \left(\frac{1}{2}\right)^2 = \frac{1}{4} = \frac{1}{S_{\triangle COD}}\]

<|ref|>equation<|/ref|><|det|>[[150, 725, 372, 760]]<|/det|>
\[\therefore S_{\triangle COD} = 4\]

<|ref|>text<|/ref|><|det|>[[162, 754, 732, 825]]<|/det|>
∵ \(S_{\triangle AOB}\) 与 \(S_{\triangle BOC}\) (不同底, 同高)
且底比为 \(OA:OC = 1:2\) 

<|ref|>text<|/ref|><|det|>[[162, 824, 777, 864]]<|/det|>
∴ \(S_{\triangle AOB} : S_{\triangle BOC} = 1:2\) ∴ \(S_{\triangle AOB} = 1\) 

<|ref|>text<|/ref|><|det|>[[135, 857, 888, 931]]<|/det|>
∴ \(S_{\triangle BOC} = 2\). 同理: \(S_{\triangle AOD}\) 与 \(S_{\triangle COD}\) (不同底, 同高)
且底比为 \(OA:OC = 1:2\) ∴ \(S_{\triangle AOD} : S_{\triangle COD} = 1:2\)
