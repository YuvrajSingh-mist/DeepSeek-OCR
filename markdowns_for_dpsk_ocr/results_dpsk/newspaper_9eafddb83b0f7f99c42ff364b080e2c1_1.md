<|ref|>text<|/ref|><|det|>[[55, 68, 655, 95]]<|/det|>
他是一个农民,他也是“民间工艺美术大师”、国家级非物质文化遗产项目永清秸秆乳制机的国家级代表性传承人；他40多年创作了80余件作品,获得中国国际民间艺术博览会山花奖、民间工艺金奖等50多个奖项

# 农家书屋

<|ref|>title<|/ref|><|det|>[[735, 81, 972, 99]]<|/det|>
# 任城农家书屋成新农民“快乐驿站

<|ref|>title<|/ref|><|det|>[[25, 105, 690, 150]]<|/det|>
# 徐艳丰：田野里走出的秸秆乳制大师

<|ref|>text<|/ref|><|det|>[[311, 154, 399, 163]]<|/det|>
□□ 本报记者 王会

<|ref|>image<|/ref|><|det|>[[8, 175, 279, 301]]<|/det|>

<|ref|>title<|/ref|><|det|>[[30, 303, 255, 311]]<|/det|>
# 徐艳丰与他女儿徐晶晶（左）、儿子徐健研究乳制技艺。张玉宇 摄

<|ref|>text<|/ref|><|det|>[[7, 314, 144, 364]]<|/det|>
永清秸秆乳制技艺至今有四十多年的历史，这项技艺的创始者叫徐艳丰，是一位地地道道从田野里走出的艺术大师。近日，记者在徐艳丰的家乡，河北省永清县刘街乡南大庄村正采访了他。

<|ref|>title<|/ref|><|det|>[[8, 371, 143, 389]]<|/det|>
# 他的作品成了国礼，秸秆乳制成了“非遗”

<|ref|>text<|/ref|><|det|>[[7, 396, 144, 471]]<|/det|>
“徐老师最大的成就，是他让草根的水清秸秆乳制成了国礼，成为了国家非物质文化遗产项目。”陪同采访的廊坊市群艺馆的王晓燕告诉记者，“1982年徐老师的作品就被当做国礼赠送给日本人，2008年永清秸秆乳制入选第二批国家级非物质文化遗产名录，2009年他成为该项目的国家级代表性传承人。”

<|ref|>text<|/ref|><|det|>[[7, 472, 144, 563]]<|/det|>
在徐艳丰家客厅的屋顶上，记者发现一只挂着的人角花灯，尽管色泽已经泛黄，但工艺精美。徐艳丰告诉记者，这是他13岁时，花费了七八个月的时间制作的。徐艳丰生于1952年，没读过书，到现在也不认识字，更不懂建筑学、力学、结构学。但是，从11岁用高架桥挂了第一只蝴蝶笔笼至今，他已创作了大小80余件作品。“徐老师的作品件件精妙、巧夺天工。”王晓燕说。

<|ref|>text<|/ref|><|det|>[[7, 564, 144, 622], [150, 314, 287, 413]]<|/det|>
回忆起他第一次创作乳制“天安门”的情景，徐艳丰依然记忆犹新。“文革”开始后，他在电影里看到毛泽东主席接见红卫兵，被宏伟的天安门城楼所吸引，他就想用高架桥拿出来。但他跟着电影跑腻了8个村子也没能看清。因为姥爷是抗日烈士，过年的时候县民政局给姥姥送来一张年画，年画的背景是鲜艳的天安门。徐艳丰却很老实。创作期间，他常常忘了喝水、吃饭，困了就睡一会儿，醒了再接着干。村里人笑话他，父母认为他不务正业，为此父亲没少打他，但他没有止步，因为秸秆乳制寄托了他全部的梦想。三年零七个月后，长两米、宽一丈，粗细长短用了70多斤高架桥天的天安门模型做好了，城墙、城楼、配楼、金水桥、华表、样样具备，观看无不交口称赞。

<|ref|>text<|/ref|><|det|>[[150, 413, 287, 488]]<|/det|>
1982年，河北省搜集民间艺术，永清县把徐艳丰的作品推荐上了去。“省里专家看了我的‘天安门'觉得非常精美，想带到日本参加省里和长野县的文化交流。”展览办了一个月，徐艳丰的《乳制“天安门"因其规模最大也最精致，展览结束的时候，征得他同意，作为礼物送给了长野县。说到这件事，徐艳丰仍然很激动。

<|ref|>title<|/ref|><|det|>[[182, 495, 255, 504]]<|/det|>
# “这个东西我不卖”

<|ref|>text<|/ref|><|det|>[[150, 512, 287, 612]]<|/det|>
1982年，从日本回国以后，徐艳丰对秸秆乳制的热情被彻底点燃。接下来他一鼓作气，用了一年零三个月的时间扎制了“佛香阁”“·佛香阁”做成后，我脑子里突然闪现了一个念头一把它送给国家。”徐艳丰说。于是他到了北京。可徐艳丰并不知道哪个部门收藏这类东西。他把“佛香阁”放到了天安门广场。精美的乳封儿上吸引了众多的目光，“当时不少人喜欢我的东西，问我卖不卖，我说不卖，这是我要送给国家的。”

<|ref|>text<|/ref|><|det|>[[168, 613, 286, 621], [293, 174, 430, 215]]<|/det|>
在一位景山学校老师的指点下，徐艳丰来到了中国美术馆。两位工作人员把他接进了美术馆，“当时美术馆馆长和一些专家都出国了，但工作人员认为我的作品非常精美，就建议我留下，回去等消息。”

<|ref|>text<|/ref|><|det|>[[293, 216, 430, 265]]<|/det|>
回到家后的徐艳丰哪里放心得下，回想家人的不解、妻子的反对、街坊邻居的冷嘲热讽，徐艳丰感到了莫大的压力。“那时真是吃不下饭，睡不着觉，20来天人瘦得皮包骨。”徐艳丰回忆说。

<|ref|>text<|/ref|><|det|>[[293, 266, 430, 356]]<|/det|>
第29天时，妻子兴冲冲地拿回一张报纸让徐艳丰看，刚看完徐艳丰就哭出了声。“那是《中国青年报》，上面登了一篇《这个东西我不卖》的文章，说的就是我在北京的那段经历，而且文章结尾处还提到“佛香阁”模型被中国美术馆收藏的事。”徐艳丰在村里引起了轰动，老父老母也终于理解了。接下来就是一批批的记者，一篇篇的报道，徐艳丰和他的乳制艺术终于为所有人肯定。

<|ref|>title<|/ref|><|det|>[[293, 363, 429, 381]]<|/det|>
# 一件作品几十万字高梁秆，每 扇窗户都能开合

<|ref|>text<|/ref|><|det|>[[293, 388, 430, 480]]<|/det|>
在徐艳丰的家里，记者发现他和女儿徐晶晶、儿子徐健的作品都是吉建筑。他历年的获奖作品如人选“中国造型艺术展"并被加拿大收藏的“黄鹤楼”获第二届民族文化博览会民间美术大展特别奖的“角楼”、获中国民间绝艺大赛金奖的“庆州白塔”等都是我国的古代建筑。“我国的古建筑博物馆，造型精美壮观，最适合用秸秆乳制来表现。"面对记者的疑问徐艳丰这样解释。

<|ref|>text<|/ref|><|det|>[[293, 480, 430, 547]]<|/det|>
“乳制这些大的古建筑，每个都要几十万字高架秆，需要两三年的时间。”徐艳丰指着“庆州白塔”告诉记者。而要把这几十万字高架秆乳制成作品绝非说说那么简单。徐艳丰祖辈是有名的木匠，到他这一辈已经是第13代，他能够成为乳制大师，正是得益于从小对房屋建造的耳濡目染。

<|ref|>text<|/ref|><|det|>[[293, 548, 430, 622], [437, 174, 573, 199]]<|/det|>
“所有乳制作品的骨架都是由六根高梁秆锁扣而成，将锁好的秆从下往上、从里往外，一层层做起来，就像盖房子。”徐艳丰介绍。“乳制的关键是尺寸要掌握好。每根秆的直径都是用游标卡尺一一校对，误差不能超过0.3毫米。高梁秆不可能个个笔直，你看现在的架柱都是笔直，那是因为在乳制之前，用酒精灯一烤，过把它烤直的。秆直且一样粗，乳制出的作品才会整齐。而且这样竖上的扇窗户和窗都能够打得开、关得上。”

<|ref|>text<|/ref|><|det|>[[437, 200, 574, 299]]<|/det|>
徐艳丰用于乳制的高梁秆都是自己家种植的，他的五六亩承包地都种了高粱。用于乳制的高梁种植要求非常严格，株距大秆就粗，株距小就细。为得到粗细不同的高粱秸秆，徐艳丰的爱人在拔苗时会特别细心地掌握籽株距。在收获的季节，一家四口更是精挑细选，四五天割完，然后晾干剥去外皮，这个过程会持续一两个月。为了使高梁秆更坚硬、颜色更好匀，徐艳丰还自己进行杂交试验，用杂交高粱乳制的古建筑模型更精美。

<|ref|>title<|/ref|><|det|>[[437, 306, 573, 324]]<|/det|>
# 未完成的“天安门全景”，原准备献礼北京奥运

<|ref|>text<|/ref|><|det|>[[437, 331, 574, 380]]<|/det|>
在徐艳丰家的客厅，有一件用塑料布精心包裹着的作品，它就是“天安门全景”。说起这件作品，它背后有个曲折的故事。“这是徐老师倾注了最多心血的作品，也是令他遗憾终身的作品。”王晓燕说。

<|ref|>text<|/ref|><|det|>[[437, 381, 574, 439]]<|/det|>
原来，2001年7月北京成功申办奥运会之后，这个朴实的庄稼汉子开始思考自己能为奥运会做点什么？“天安门是祖国的象征，没有什么建筑比它更能代表我们强大的祖国了。我要扎制‘天安门全景”，为北京奥运会献上最深的祝福！”

<|ref|>text<|/ref|><|det|>[[437, 440, 574, 489]]<|/det|>
然而，就在徐艳丰紧张赶制这件奥运献礼作品的时候，2002年中秋节，他被病魔击倒了，经医生检查，他得了尿毒症，而且是晚期。在与死神搏斗的日子里，家庭的经济困难和昂贵的医药费用，让这位乳制大师几度欲放

<|ref|>text<|/ref|><|det|>[[580, 174, 717, 249]]<|/det|>
弃治疗。在永清县委县政府和各级文化部门的多方努力和支持下，他成功完成了两次换骨手术。“只要还有一口气，我就会把‘天安门全景”做完，而且比第一件做得更精致，更完美，为我的艺术生涯划上一个完美的句号。”但由于病重和经济拮据，原定2008年5月完工、为北京奥运献礼的“天安门全景”至今未能完成。

<|ref|>title<|/ref|><|det|>[[590, 257, 707, 266]]<|/det|>
# 如今无力创作，唯愿技艺传承

<|ref|>text<|/ref|><|det|>[[580, 273, 717, 323]]<|/det|>
采访的时候，徐艳丰拄着一只拐杖，他告诉记者，除了患有尿毒症，现在他还患有腰椎间盘突出症，因为手会发抖，去年开始，他已不再进行扎制创作。“女儿晶晶和儿子徐健都制作乳制，技艺已经超过我了。”

<|ref|>text<|/ref|><|det|>[[580, 323, 717, 397]]<|/det|>
“因为秸秆乳制周期长，难度大，完成一件作品往往需要几个月甚至几年的时间，也赚不了什么钱，因此没有多少人愿意学习。”徐艳丰说。他病了以后，为了传承此项技艺，女儿中断了中专的学业，儿子考上了医科大学也没有去报到，“这个无奈的选择已成为徐艳丰老师心中永远的痛。”陪同采访的王晓燕对记者说。

<|ref|>text<|/ref|><|det|>[[580, 398, 717, 456]]<|/det|>
县里对徐艳丰做了最大的支持，他们的扎制作品常常被作为礼物送给县里的客人，但是家境的法绍基还是能一眼看到，邻居家都盖了高大的新房，只有他家的还是旧房子。因为家里经济条件差负担重，29岁的女儿晶晶和26岁的儿子徐健到现在还没有成家。

<|ref|>text<|/ref|><|det|>[[580, 456, 717, 489]]<|/det|>
“我现在最希望的就是有更多的人学习秸秆扎刻艺术，让这一技艺传承下去。”说到这个话题，徐艳丰的神情充满期盼。

<|ref|>image_caption<|/ref|><|det|>[[502, 615, 680, 623]]<|/det|>

<|ref|>text<|/ref|><|det|>[[732, 106, 870, 165]]<|/det|>
秋果收尽，时至冬日。在山东省济宁市任城区长沟镇孙街村文化大院的农家书屋里，每天都汇聚了不少果来翻阅书籍报刊。孙街村是葡萄种植专业村，全村家家户户都种葡萄。农家书屋建成至今，这里成为了果农们学习交流种植经验的“快乐驿站”。

<|ref|>text<|/ref|><|det|>[[732, 166, 870, 224], [876, 106, 995, 225]]<|/det|>

<|ref|>title<|/ref|><|det|>[[744, 538, 980, 553]]<|/det|>
# 湘西自治州推进“民族文化进机关”

<|ref|>text<|/ref|><|det|>[[732, 560, 870, 584]]<|/det|>
