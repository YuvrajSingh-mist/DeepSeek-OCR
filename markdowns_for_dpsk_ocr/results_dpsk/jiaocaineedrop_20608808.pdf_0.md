## 一、单选题：本题共10小题，每小题5分，共40分。在每小题给出的四个选项中，只有一项是符合题目要求的.  

<|ref|>text<|/ref|><|det|>[[78, 440, 280, 475]]<|/det|>
1. 已知函数 \(f(x) = \int_{0}^{x}e^{\cos t}dt,g(x) = \int_{0}^{\sin x}e^{t^{2}}dt\) 则  

<|ref|>text<|/ref|><|det|>[[88, 480, 430, 499]]<|/det|>
A. \(f(x)\) 是奇函数， \(g(x)\) 是偶函数 
B. \(f(x)\) 是偶函数， \(g(x)\) 是奇函数  

<|ref|>text<|/ref|><|det|>[[88, 506, 410, 525]]<|/det|>
C. \(f(x)\) 与 \(g(x)\) 均为奇函数 
D. \(f(x)\) 与 \(g(x)\) 均为周期函数  

<|ref|>text<|/ref|><|det|>[[70, 533, 476, 586]]<|/det|>
[答案]：C[解析]：【解析】由于 \(e^{\cos t}\) 是偶函数，所以 \(f(x) = \int_{0}^{x}e^{\cos t}dt\) 是奇函数，又 \(g^{\prime}(x) = e^{(\sin x)^{2}}\cos x\) 是偶函数，所以 \(g(x)\) 是奇函数.故选C.  

<|ref|>text<|/ref|><|det|>[[78, 591, 475, 650]]<|/det|>
2. 设 \(P = P(x,y,z)\mathbb{D}Q = Q(x,y,z)\) 均为连续函数， \(\sum\) 为曲面 \(Z = \sqrt{1 - x^{2} - y^{2}}(x\leqslant 0,y\geqslant 0)\) 的上侧，则 \(\iint_{\Sigma}P d y d z + Q d z d x =\)  

<|ref|>text<|/ref|><|det|>[[88, 653, 395, 730]]<|/det|>
A. \(\iint_{\Sigma}\left(\frac{x}{z} P + \frac{y}{z} Q\right)d x d y\) B. \(\iint_{\Sigma}\left(-\frac{x}{z} P + \frac{y}{z} Q\right)d x d y\) C. \(\iint_{\Sigma}\left(\frac{x}{z} P - \frac{y}{z} Q\right)d x d y\) D. \(\iint_{\Sigma}\left(-\frac{x}{z} P - \frac{y}{z} Q\right)d x d y\)  

<|ref|>text<|/ref|><|det|>[[70, 735, 375, 765]]<|/det|>
[答案]：A[解析]：【解析】转换投影法， \(z = \sqrt{1 - x^{2} - y^{2}}\) \(\frac{\partial z}{\partial x} = -\frac{x}{z},\frac{\partial z}{\partial y} = -\frac{y}{z}\)  

<|ref|>equation<|/ref|><|det|>[[196, 777, 377, 813]]<|/det|>
\[\iint_{\Sigma}P d y d z + Q d z d x = \iint_{\Sigma}\left(\frac{x}{z} P + \frac{y}{z} Q\right)d x d y\]  

<|ref|>text<|/ref|><|det|>[[88, 824, 120, 840]]<|/det|>
故选A.  

<|ref|>text<|/ref|><|det|>[[78, 848, 312, 881]]<|/det|>
3. 已知幂级数 \(\sum_{n = 0}^{\infty}a_{n}x^{n}\) 的和函数为 \(\ln (2 + x)\) ，则 \(\sum_{n = 0}^{\infty}n a_{2n} =\)  

<|ref|>text<|/ref|><|det|>[[88, 884, 404, 916]]<|/det|>
A. \(\frac{1}{6}\) B. \(\frac{1}{3}\) C. \(\frac{1}{6}\) D. \(\frac{1}{3}\)  

<|ref|>text<|/ref|><|det|>[[545, 80, 857, 110]]<|/det|>
[答案]：A[解析]：【解析】方法1： \(\ln (2 + x) = \ln 2\left(1 + \frac{1}{2} x\right) = \ln 2 + \ln \left(1 + \frac{1}{2} x\right)\)  

<|ref|>equation<|/ref|><|det|>[[696, 115, 800, 155]]<|/det|>
\[= \ln 2 + \sum_{n = 1}^{\infty}(-1)^{n - 1}\frac{\left(\frac{1}{2} x\right)^{n}}{n}\]  

<|ref|>text<|/ref|><|det|>[[561, 167, 950, 255]]<|/det|>
所以， \(a_{n} = \left\{ \begin{array}{c c}{{\ln 2,}} & {{n = 0}}\\ {{(- 1)^{n - 1}\frac{1}{n^{2n}},}} & {{n > 0}} \end{array} \right.\) 当 \(n > 0,a_{2n} = - \frac{1}{2n\cdot2^{2n}}\) 所以， \(\sum_{n = 0}^{\infty}n a_{2n} = \sum_{n = 1}^{\infty}n a_{2n} =\) \(\sum_{n = 1}^{\infty}n\cdot \left(-\frac{1}{2n\cdot2^{2n}}\right) = -\sum_{n = 1}^{\infty}\frac{1}{2^{2n + 1}} = -\frac{\left(\frac{1}{2}\right)^{3}}{1 - \frac{1}{4}} = \frac{1}{6}\) 故选A.方法2：  

<|ref|>equation<|/ref|><|det|>[[630, 268, 860, 308]]<|/det|>
\[\ln (2 + x)]^{\prime} = \frac{1}{2 + x} = \frac{1}{2(1 + \frac{x}{2})} = \frac{1}{2}\sum_{n = 0}^{\infty}(-1)^{n}\left(\frac{x}{2}\right)^{n}\]  

<|ref|>equation<|/ref|><|det|>[[625, 320, 864, 363]]<|/det|>
\[\ln (2 + x)]^{\prime} = \frac{1}{2 + x} = \frac{1}{2(1 + \frac{x}{2})} = \frac{1}{2}\sum_{n = 0}^{\infty}(-1)^{n}\left(\frac{x}{2}\right)^{n}\]  

<|ref|>equation<|/ref|><|det|>[[625, 366, 864, 408]]<|/det|>
\[S(0) = C = \ln (2 + 0) = \ln 2\]  

<|ref|>text<|/ref|><|det|>[[625, 410, 800, 456]]<|/det|>
所以， \(a_{n} = \left\{ \begin{array}{c c}{\ln 2,} & {n = 0}\\ {(- 1)^{n - 1}\frac{1}{n2^{n}},} & {n > 0} \end{array} \right.\)  

<|ref|>text<|/ref|><|det|>[[561, 460, 910, 500]]<|/det|>
所以， \(\sum_{n = 0}^{\infty}n a_{2n} = \sum_{n = 1}^{\infty}n a_{2n} = \sum_{n = 1}^{\infty}n\cdot \left(-\frac{1}{2n\cdot2^{2n}}\right) = -\sum_{n = 1}^{\infty}\frac{1}{2^{2n + 1}} = -\frac{\left(\frac{1}{2}\right)^{3}}{1 - \frac{1}{4}} = -\frac{1}{6}\) 故选A.  

<|ref|>text<|/ref|><|det|>[[545, 500, 780, 520]]<|/det|>
4. 设函数 \(f(x)\) 在区间 \((-1,1)\) 上有定义，且 \(\lim_{x\to 0}f(x) = 0\) ，则  

<|ref|>text<|/ref|><|det|>[[561, 525, 900, 590]]<|/det|>
A.当 \(\lim_{x\to 0}\frac{f(x)}{x} = m\) 时， \(f^{\prime}(0) = m\) B.当 \(f^{\prime}(0) = m\) 时， \(\lim_{x\to 0}\frac{f(x)}{x} = m\) C.当 \(\lim_{x\to 0}f^{\prime}(x) = m\) 时， \(f^{\prime}(0) = m\) D.当 \(f^{\prime}(0) = m\) 时， \(\lim_{x\to 0}f^{\prime}(x) = m\)  

<|ref|>text<|/ref|><|det|>[[545, 592, 950, 650]]<|/det|>
[答案]：B[解析]：【解析】因为 \(f^{\prime}(0) = m\) ，所以 \(f(x)\) 在 \(x = 0\) 处连续，从而 \(\lim_{x\to 0}f(x) = f(0) = 0\) ，所以 \(\lim_{x\to 0}\frac{f(x)}{x} = \lim_{x\to 0}\frac{f(x) - f(0)}{x - 0} = m\) ，故选B.  

<|ref|>text<|/ref|><|det|>[[545, 654, 950, 704]]<|/det|>
对于A选项， \(\lim_{x\to 0}\frac{f(x)}{x} = m\) ，推不出来 \(f^{\prime}(0) = m\) ；对于C选项， \(f^{\prime}(x)\) 在 \(x = 0\) 处不一定连续；对于D选项， \(f^{\prime}(x)\) 在 \(x = 0\) 处极限未必存在.
