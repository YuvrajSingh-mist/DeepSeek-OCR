10. 引导结果状语从句的从属连词有: so that (结果是) 和 so / such ... that ... (如此...以至于) 等. 

<|ref|>text<|/ref|><|det|>[[130, 175, 780, 207]]<|/det|>
It was very cold, so that the water in the bowl froze. 

<|ref|>text<|/ref|><|det|>[[130, 207, 740, 238]]<|/det|>
It is such a heavy box that nobody can move it. 

<|ref|>text<|/ref|><|det|>[[85, 238, 938, 300]]<|/det|>
11. 引导比较状语从句的从属连词有: as ... as (与...一样), not as / so ... as ... (不及, 好不上), 和 that (此) 等. 

<|ref|>text<|/ref|><|det|>[[130, 302, 562, 333]]<|/det|>
7 know you better than she does. 

<|ref|>text<|/ref|><|det|>[[130, 335, 533, 366]]<|/det|>
He works as carefully as she. 

<|ref|>text<|/ref|><|det|>[[130, 366, 540, 397]]<|/det|>
7 can't run as / so fast as you. 

<|ref|>text<|/ref|><|det|>[[85, 394, 830, 426]]<|/det|>
12. 引导名词性从句的从属连词有: that 和 it / whether (是否) 等. 

<|ref|>text<|/ref|><|det|>[[130, 426, 835, 490]]<|/det|>
We know that the earth goes around the sun. (宾语从句)
Whether he'll go there hasn't been decided. (主语从句) 

<|ref|>text<|/ref|><|det|>[[130, 486, 936, 549]]<|/det|>
[注意] 在时间状语从句和条件状语从句中, 如果主句是将来时, 从句要用一般现在时.
