<|ref|>equation<|/ref|><|det|>[[180, 816, 360, 857]]<|/det|>
\[\sum_{n = 1}^{\infty}\frac{(-1)^{n - 1}}{n(2n + 1)} = \sum_{n = 1}^{\infty}\frac{(-1)^{n - 1}}{n} +2\sum_{n = 1}^{\infty}\frac{(-1)^{n}}{2n + 1}.\]

注意到  

<|ref|>equation<|/ref|><|det|>[[190, 884, 355, 923]]<|/det|>

<|ref|>equation<|/ref|><|det|>[[630, 175, 864, 214]]<|/det|>
\[f^{\prime}(x) = \sum_{n = 1}^{\infty}(-1)^{n}x^{2n} = -\frac{x^{2}}{1 + x^{2}} = \frac{1}{1 + x^{2}} -1,x\in (-1,1).\]
