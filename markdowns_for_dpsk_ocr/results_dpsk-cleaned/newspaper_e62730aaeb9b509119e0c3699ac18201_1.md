# 河南省确山县“148”法律服务所所长张天佑为农民工维权纪实

55岁的张天佑在河南省确山县从事基层法律服务工作已整整21个年头。特别是从2000年为我农民工维权服务以来，农民工走到哪里，他就服务到哪里，足迹遍及大半个中国，总行程超过20万公里。为确山县及周边区县的54名伤亡农民工(其中死亡20人、伤残34人)维权索赔，总金额超过400万元。张天佑先后多次被省、市、县评为先进工作者，2007年被河南省政法委表彰、驻马店市委政法委评为“践行社会主义法治理念先进个人”；先后被评为2009“感动中原”十大年度人物候选人之一和2009“感动天中”十大年度人物。

# 为农民工维权，心甘情愿，不辞千辛万苦

作为一个基层法律服务工作者，张天佑同志深深感到：关注和帮助伤残农民工这一弱势群体，为外出打工的农民工维权，为农村劳务输出提供法律服务，是法律服务工作者义不容辞的责任。张天佑年过半百，身患高血压、糖尿病等多种疾病，但他心里装着农民工，从不考虑自己。他与当事人一起出差，给为老方拆迁包装是零成本，住户小改，小缝补。2003年，他为确山县竹沟镇珠楼村五眼井村的张传新到北京维权索赔。为了节约开支，他们冒着飞机“走专车”网暴乘客成本，每餐馒头缺辣酱，吃罢喝半天，直到得到赔款。他曾经为确山县竹沟镇的曹川军工作索赔，三赴霸州，也曾为确山县瓦岗乡刘庄村村屯工作索赔，五赴青岛。不管是严冬还是酷暑，他从来没叫过一声苦。有些同志问张天佑经常在外面跑否否？张天佑在其《五赴青岛》的诗中这样写道：“虽经千辛万苦，粒粒都化云和烟。我为霸州者伸正义，千辛万苦心也甜。”

2008年5月，确山县蛟峰镇庞阁村农民工张润民在广东省塘岭破工，发生工伤后长期失工。在塘岭十多日，张天佑冒着高温、顶烈日，天天往公司跑，有时一天跑两趟没处理。最后，公司见顶不过，拖不过，终于在11.5万元赔偿协议上签下了字。7月23日，张天佑与张润民登车返回。7月24日，张天佑刚刚回到确山，就接到了北京市朝阳区劳动仲裁打来电话，通知张天佑与伤残工人邓海龙于7月26日到北京朝阳区劳动仲裁出院。张天佑仅仅在家中睡了一觉，就又登上起点的列车。张天佑与邓海龙到北京后，在朝阳区劳动仲裁庭上，张天佑为邓海龙的申诉请求得到了仲裁庭的支持。公司老乖地在邓海龙处伤残赔偿协议上签了字，邓海龙得到56000元赔偿。张天佑满怀胜利的喜悦写下《汗洒维权路》一诗：“昨伏南粤水，今品燕京牌。增城索赔案，朝阳把案立。不怕老板横，仗剑行千里。汗洒维权路，只为申正义！”

# 为农民工维权，尽职尽责，想尽千方百计，竭尽全力

张天佑为农民工维权中，他牢记服务宗旨和法治理念，总是在农民工最需要的时候，及时出现在农民工的面前。

2005年3月30日上午，张天佑刚到竹沟镇肖庄村坳村路上三岔壶茶剧团到家中，刘店镇刘店村村民鲁淑娥拨打“148”电话，向“148”求助。称其丈夫刘银奎在安徽淮南煤矿打工，当日凌晨在井下作业，发生事故死亡。欲带来“148”为其提供法律援助，与其家人一起去淮南代理索赔。张天佑拨到电话，顾不得刚从北京归来的疲劳，就与妻子徐丽广、到淮南东岳大队与肖华队交往多，该单位在赔偿标准上一直坚持低标准。在找情况下，张天佑代表死者家人向肖华队声明索赔主张：一、不花人，无理取闹，不在政策和法律规定范围之外提要求。二、用人单位必须依法足额赔偿。在张天佑有理有据有节价的争取下，双方将对限金的工亡待遇议定到25万元。然后张天佑用足用活政策与法律，根据民法相关规定，向用人单位又提出10万元的精神损害赔偿。由于张天佑言之有据，肖华队已无法拒赔，最终为刘银奎家人争取到30万元的足额赔偿。这笔赔偿不仅为刘银奎家人今后生活提供了可靠的经济保障，还创下该县农民工索赔之最。

# 为农民工维权，不畏艰难万险，与违法 企业斗智斗勇

张天佑在为农民工维权中，不论苦再难，他都毫不退却，虽然五十多岁，身患多种疾病，但不管是严冬酷暑，案发在哪里，他就奔向哪里。

2007年元月，时值隆冬。确山县任镇倪庄村村民梅珠在新疆乌鲁木齐回工公司拉运建筑材料时，遭遇车祸死亡。春节临近，张天佑听到梅珠妻子邢花的哭诉之后，元月日便与邢氏姐弟起程赴疆找到用工单位新疆国建公司索赔，该公司推托说：“梅珠死于交通事故，用工单位无责任，不予赔偿。”张天佑据理力争道：“梅珠是为公司拉运建筑材料遭车祸死亡的，梅是公司的一名员工，公司无事故责任，但有法定抚恤义务。有义务对境中的工家属予以救助。”在谈判期间，由马鲁木齐冰大雪地，天气零下十几度，张惠上重量，到医院输液，为了不误谈判约定，张天佑输液，就匆忙往公司赶，坚持谈判。在张天佑据理力争下，最终与用人单位达成2.5万元的补偿协议，使邢花在法院诉讼未果的情况下得到救助2.5万元。钱虽然不多，但却解决了邢花母女过春节的燃眉之急，同时也为春节后邢花再赴新疆诉讼，准备了费用。在返回的车上，张天佑写下《赴新疆》一诗，记载与新疆白道公司谈判的过程：“白道交涉无结果，五次再谈可商量。有理有据报室稳，稳扎稳打不慌张。”诗的最后，他满怀胜利地喜悦写道：“法律武器是个宝，我福法宝走四方！”

几年来，张天佑走遍全国各地，与那些大气粗粗的大大小小的违法企业老板打交道，受尽了人们的歧视、刁难和威胁。张天佑毫不气，运用法律武器，沉着、冷静地与他们斗智斗勇。他和一个个违法企业老板、包工头谈判不下数百次，再难除他从未打过退堂鼓，从未向邢部势力低头过，直到为农民工依法讨回公道为止。

# 为农民工维权，不计较吉利与否，为尽职十进久葬场

确山县“148”在法律服务上，向社会一承诺：“当您的合法权益受到侵犯时，‘148'将为您竭诚提供优质的法律服务。”张天佑作为“148”主任，他是这样说的也是这样做的。2007年5月17日，确山县留庄镇汪庄村马庄组64岁的单身汉李营在北京丰台一建筑工场打工，被一重物砸伤头部，经抢救无效死亡。其弟弟金太太起诉“148"求助，张天佑接受委托后，与李一同前往，到京之后，张天佑克服用人单位设置的重重障碍，历时21天，最终使用人单位作出14.7万元的赔偿。《协议》签订后第二天，张天佑又协助其弟弟去到医院太平间将李营户体取出，送到房山区殡仪馆火化。当天上午，张天佑与李金太太打一同乘车到房山殡仪馆，一直将李营的尸体送到火化炉前。张天佑竭诚为当事人服务，使李金太太悲痛得直掉眼泪。

2009年8月4日，确山县瓦岗乡洋沟村农民工张富安在保证无“一建筑工程集团公司丁死亡，张天佑接受张富安家人的委托，前往河北保定维权。到达保定后，经过与公司三天的谈判，为张富安家人承担赔偿43万。赔偿协议达成后，张天佑又与其家属将张富安的尸体护送保定殡仪馆火葬，直至办完张富安的后事，才与张富安家人返回。在保定殡仪馆为张富安办理后事，是张天佑为二十位工代的农民工中的第十位农民工选择办理后事。在常人眼中，殡仪馆、火葬场是个不吉利的地方，但是张天佑为农民工服务，从不计较这些。

# 为农民工维权，不分白天黑夜，足迹遍及大半个中国

在张天佑的日历上没有星期六和节假日，不分白天和晚上，只要哪里发生纠纷，他就会不辞劳苦地出现在哪里。在为外出农民工维权的道路上，他的足迹遍及大半个中国，切切实实保护了农民工的合法权益。张天佑以优质的服务，将确山县“148"打造成为“人民满意的政法单位”，使确山县“148"深深地扎根于群众之中，成为确山县司法行政系统中的一面旗帜。

20多年来，张天佑在平凡的法律服务岗位上，把法律服务工作做到群众的心窝里。他用自己爱岗敬业的精神和优质的服务赢得了党和人民的赞誉，走出了一条党做的法律服务农民工的正义之路。

今年被评为河南省汤阴县“十佳共产党员标兵”的黄治东，是该县城关镇李孔村一名普通村民。他从20多岁时始到养鸡场打工赚钱，靠着踏踏实实的埋头工作，认认真真的钻研技术，虚心向身边人学习、请教，终于练就一身养鸡本领。打工十年后他以此为本钱办起了自己的养鸡场，不仅成为当地拥有固定资产500万元、年产值1400万元的养殖大户，也成为远近闻名的“治东养殖专业合作社"理事长。说起他的创业经历，黄治东感慨万分：“打工不光是又苦又累，它也是一所大学校，只要你在这所学校里干锤百炼，它同样会给你一笔无形的财富，让你受益终生！”

# 任城区“阳光培训”助推农民工创业就业

12月14日，山东省济宁市任城区举办的“艳阳工程”第三期技能培训班在该区石桥镇圆满结束。200多人参加了涉及电脑、市场营销、电气焊三个专业技能培训班，通过了培训结业考试。自此，该区举办的第三期“艳阳工程"技能培训班全部结束。其中，137名农民工拿到了高级证书，63名农民工拿到了中级证书，158人取得了初级证书。

今年以来，任城区把农民工技能培训作为大事来抓，投资300余万元在苍镇（街道)驻地建设完善了培训基地。以现有教育培训机构为主渠道，发挥多种教育培训资源的作用，积极引导鼓励教育培训机构与人力资源开发机构建立合作伙伴关系，通过签订培训订单或创业协议，约定双方责任和权益，实现培训与创业的良性互动。先后举办农村绿色证书、农村干部和农村后备干部培训班、农业技术培训班、农村富余劳动力职业技能培训班等50余班次。共发放绿色证书3000人次、培训农村干部300人次、农村后备干部1000人次，培训农业技术人员10000余人次，农村富余劳动力职业技能培训5000人次。

上半年，随着该区石桥镇万亩荷花生态园的不断完善，餐饮服务业用工需求加大，石桥镇大力开展了餐饮、礼仪服务等订单培训，实现就地安置200余名农民工。

目前，该区“艳阳工程”三期培训班460名农民工，已经在计算机、电气焊、车工、电工等专业上提高了实用技能，获得了国家承认的专业证书，增强了创业就业的资本。

近年来，该区还对贫困家庭子女实施技能扶贫，让其免费到技工学校参加技能培训，先后选拔500多名贫困生进入了扶贫技校进行学习，每人补贴70\~220元，已培训8000余人次。对获得高级证书和中级证书的学员，政府再分别给予1600元和800元的培训补贴，提高了农民工参加技能培训的积极性。

# 叙永县：矿山农民工养育老保险

日前，四川省叙永县出台文件，要求矿山企业必须为农民工办理养老保险。

近年来，叙永县社保工作得到长足发展，但矿山企业农民工的社会保障却是一个空白，大部分在矿山企业工作的农民工，基本上没有参加养老保险。为维护矿山企业职工的合法权益，壮大综合保缴费面，该县采取多种措施，促进矿山企业为农民工办理

养老保险。

据叙永县社保局有关人员介绍，该县建立了劳动监管制度，从检查评估、稽核工作入手，加大对养老保险政策的宣传力度，同时，要求各矿山企业必须为农民工购买养老保险，并出台相关优惠政策，调动矿山企业为农民工缴纳养老保险费的积极性。截至目前，县里已有11家矿山企业为698名农民工办理了养老保险。曾周海

的工作，黄治东一干就是6年。打工开始之时，黄治东只是想把自身工作干好，干着干着，他对养鸡的兴趣越来越浓。随之，再考虑就不觉其苦，黄治东把全部精力都投入到养鸡之中。黄治东回顾说：当饲养员确实很艰辛，寒冬腊月，每天往返奔波于零下10多度的室外和零上20多度的鸡舍，摸变的温差和任的操劳，常常使人的体力严重透支。疲劳过度的他，一次上厕所，竟然晕倒了。当他跌跌撞撞回到鸡舍后，四肢瘫软失去了知觉，醒来后他才知道自己在医院里已经昏迷了整整三天三夜。1994年，种鸡场暴发鸡新城疫，短短四天，种鸡成批死亡，特别是他负责的6号鸡舍5000多只成性种鸡只剩下2000多只，造成的直接经济损失高达40万余元。恰在此时，黄治东负责紧张的7号鸡舍，5000只出窝10余天的种鸡活，正处于免疫力低下的高危期，稍有疏忽就有可能被这批种鸡感染新城疫，后果不堪设想。面对压力，刚刚新婚三天的黄治东两个多月吃住都在鸡场，精心饲养、用心管理、科学防疫，不敢有任何懈意，终于使这批种鸡健健康康地度过了高危期，为鸡场的重生留住了希望，避免了30多万元的经济损失，创造了60余万元的经济收人。两个月下来，原本白白胖胖的黄治东却变得又黑又瘦。

我们每个人都有不同程度的自卑感，只是表现形式不同而已。当我们面对一个无法处理的问题或绝无力决解的问题时，就会体会到这种自卑情绪。没有人能长期地忍受自卑感，他一定会采取某种行动来解除自己的紧张。但先当看到，当人们发现自己所处的位置是可以加以改善时，面对更高的愿望，会定生产内在动力。因此，自卑感往往也是推动人不断地努力去取得成功、消除紧张感的因素。

2009年7月北京发生了一起在校大学生捡拾烟头案，案件很快被侦破，留给我们一些启示。繁荣来自江西农村，家庭经的种鸡育雏期成活率98.9%，高于平均水平5个百分点，产蛋率86%，高于平均水平7个百分点。他赢得了鸡场老板的赞赏，凭着自己的才干先后担任了班长、生产副科长、生产区长，多次被评为模范村长，并在1995年养鸡场农民工中第一个加入了中国共产党，为自己的打工经历画上了一个骄人的符号。

# 困难和问题是成长的“老师”

黄治东肯动脑筋、勇于创新，打工期间常常苦苦思冥想，小打小闹地解决了种鸡饲养过程中的一些实际问题。为了解决养鸡防疫难问题，在征得养鸡老板同意的情况下，黄治东对传统的群体防疫方法进行改革，率先采用个体免疫法，每只鸡施行静脉注射防疫针，保证了防疫效果，避免了传统防疫引起的防疫不均现象，提高有了育雏期鸡苗的成活率。他先后改进了鸡舍供暖系统、隔离系统、通风系统。经改造的多功能供暖系统实现了节能、安全、环保，并有效延长了采暖炉、送热管的使用寿命，增加了鸡舍温度，保证了供热均匀性。他设计的地下水箱沉降基准系统跑、环保，又无污染。黄治东凭着打工期间积累的经验和技术，后来在自己改修的鸡舍内，他将过去的几十天前进行整合，实现了水上、料上自动化，通风、采光自动化，温度控制自动化，大大降低了养殖成

# 创业坎坷，难阻挡前行脚步

“打工也是一所大学校”，黄治东在这所学校里是一个合格、优秀的学生，同时这所学校也使黄治东积累了丰富的养鸡经验和宝贵的人生阅历。2000年，畴曙满志的黄治东开始创业之路。这一年他便养了4批肉鸡，净赚了1.6万元。初次创业的成功使他备受鼓舞，然而天有不测风云，2003年，一场突如其来的非典让他几乎倾家荡产，仅仅两个月黄治东净赔了18万元。但黄治东靠着当年打工期间练就的在困难和挫折面前百折不回的意志，顶住了压力、克服了困难，坚持了下来。非典过后，他不但把前几个月赔的18万元挣了回来，而且又净赚6万元。

为了使乡亲们走向共同富裕的道路，2008年黄治东联系了5家养殖企业经营户成立了“治东养殖合作社”，在短短三年间，“治东养殖合作社"发展涉及养殖产业中的孵化、育苗、种鸡、饲料、防疫、分割、销售等全部链条，覆盖四省五市18个县区，联系户达到500余户，带动了整个徽北地区的养殖产业。黄治东经常说，“如果我没有当年的打工经历，如果我在打工期间只是混日子、磨洋工，那么，今天的我是个什么样子还说不定呢！”

在当今竞争激烈的年代，许多年轻人以玩世不恭的态度对待工作，他们频繁跳槽，觉得自己工作是在出卖劳动力；他们蔑视敬业精神，嘲讽忠诚，将其视为老板盘剥、愚弄下属的手段。

现代管理学普遍认为，老板和员工是一对矛盾的统一体，从表面上看起来，彼此之间存在着对立性——老板希望减少人员开支，而员工希望获得更多的报酬。但是，在更高的层面上，两者又是和谐统一的——公司需要敬业和有能力的员工，业务才能进行；员工必须依赖公司的业务平台才能获得物质报酬和满足才能施展及精神需求。因此，双方实为相互依存的整体。

敬业是职场中最应该值得重视的美德。只有每个员工敬业，才能发挥出团队的力量，才能拧成一股绳，劲儿往一处使，推动企业走向成功。一个公司的生存依靠少数员工的能力和智慧，但更需要绝大多数员工的敬业和敬业。

老板在用人时不仅只看重个人能力，更看重个人品质，而品质中最关键的就是忠诚度。在这个世界上，并不缺乏有能力的人，那种既有能力又忠诚的人才是每一个企业求企的理想人才。对门号信任一个能力差一起却又够敬业的人，而不愿意用一整个三靠口、视敬业为无物的人，哪怕他能力再差。如果你是老板，你肯定也会这样做。

当你的敬业精神增加一分，别人对你的尊敬也会增加一分。不管你能力如何，老板会乐意你在身上投资，给你培训的机会，提高你的技能。因为他认为，你是值得他信赖和培养的。因为他相信，当公司面临危难的时候，你会和公司同舟共济。

老板是人，也有缺点，也可能因为大主观而无法对你做出客观的判断，这个时候你应该学会自我肯定。只要你竭尽所能，做到问心无愧，你就在不知不觉中提高了自己的能力，争取到了未来事业成功的砝码。

忠诚是一种职业敬业精神。如果你选择了某一个工作，那就真诚地、负责任地干吧，你的事业之帆已在启航。
