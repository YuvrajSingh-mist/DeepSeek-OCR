# 安徽全力投钱见实效

本报讯(记者 夏树)9月下旬以来，安徽淮北沿淮地区连续两个多月没有有效降雨，是1961年以来第三个雨年份，而且持续气温偏高，造成旱情不断加剧，对小麦播种和正常生长造成极为不利影响。面对严峻形势，安徽全省上下认真贯彻落实农业部和省委、省政府的决策部署，强化组织发动，调动基层和农民积极性，全力抗旱浇灌，抗旱保苗取得了阶段性成效。

旱情出现后，安徽省委省政府高度重视，省委书记张宝顺在省委八届十三次全会、省长王三运在省政府第十二次全体会议上，对抗旱保苗工作提出明确要求，并多次作出批示，要求想尽一切办法抓好抗旱保秋种工作。省委常委、副省长赵树丛深入抗旱

保苗一线检查指导。

安徽省农委把抗旱保苗作为工作重点，要求全省各级农业部门和农技人员，全力以赴指导帮助农民抓好适晚秋种工作。

为推动抗旱保苗顺利进行，安徽切实加大了物资保障力度。11月15日安徽省防指下达2420万度抗旱用龟；12月1日，省财政紧急下达2000万元抗旱经费，支持受旱地区抗旱保苗、浇灌补墙；在受旱地区提前启动明年抗旱机具购置补贴，从已下达各地的明年第一批农机购置补贴资金中安排2000万元专门用于抗旱机具补贴。对受旱地区预拨3亿元农资综合补贴，扶持农民抗旱保苗。受旱地区的市县政府加大抗旱资金支持。截至12月10日，小麦主产

区累计投入抗旱资金8.6亿元，其中县级财政资金7159万元。安徽省政府成立了3个抗旱保苗工作督查组，赴受旱地区开展督查。省防指及时组织会商，定期发出旱情预警，并建立了抗旱保苗进度每天通报制。省农委组织专家制定了抗旱保苗技术意见，开展了抗旱保苗技术培训，组织了抗旱保苗技术宣传；并成立了1个抗旱保苗服务组，对各地抗旱保苗工作实行全程督导；安排41个委属单位对口联系31个重点县，实行全程指导。全省有300多名农技人员包片驻点，全力推进抗旱保苗。

截至目前，截至12月10日，12月初仍然受旱的近1400万亩小麦基本浇灌一垄，有些田块浇了2遍。抗旱保苗工作取得阶段性成效，为实现壮苗稳墒奠定了基础。目前，除极少数晚播春性品种外，基本全部出苗，长势接近常年。据安徽省农委副主任王华介绍，安徽全省农业部门将进一步加强技术指导，抓好冬前田间管理，促进苗情转化，努力实现壮苗安全越冬，为夺取明年夏粮丰收奠定坚实基础。

# 山东全面动员显威力

本报讯(记者 吕兵兵)今年9月下旬以来，山东旱情发展迅速，农作物受旱面积已近2600万亩，占全省小麦播种面积的48%。面对严峻的旱情，山东农业、财政、水利、电力等部门及时行动，全力抗旱。截至目前，山东已有230多万人投人抗旱一线，投入抗旱资金超过1.5亿元，累计灌溉小麦近2000万亩次，占小麦播种面积的35%以上。

山东省农业厅先后两次组织省农业专家顾问团小麦分团和省农业技术推广总站的小麦专家分赴济宁、临沂、淄博、滨州、德州、聊城、泰安、菏泽、潍坊等9市，对小麦旱情进行考察，并于12月1日在济南召开了全省小麦苗情专家会商会议。针对发现的干旱、旺长、病虫草害等问题，专家组提出了三项指导意见：一是把浇好冬水、抗旱保苗作为当前麦田管理的首要任务。二是增加抗旱专项资金投入，为抗旱保苗提供强有力的物质保障。三是以抗旱为重点，做好冬前及冬李的麦田管理技术指导工作，针对今年冬前温度偏高，冬前生长量较大的实际情况，各地要因地制宜，做好中耕镇压、病虫草害防治、浇后划锄等工作。

同时，山东还组织万名农业科技人员，深人田间地头指导抗旱工作，利用电视、广播、发放明白纸等方式，指导农民抗旱保苗。

12月17日，山东省财政厅紧急向临沂、日照等10个小麦播种面积大、旱情严重的市，拨付特 large 抗旱补助资金2000万元，支持夏粮区开展生产自救。

面对严重旱情，国家电网山东电力集团公司及时组织两省单位成立抗旱保服务小分队，携带配电变压器、电表箱等物资，深入农村田间地头，架设灌溉线路，开辟临时用电申请绿色通道，及时上门为农民办理农业生产临时用电手续，全力支援抗旱，确保农业灌溉和入春饮水用电。

# 全国农村固废项目多超额完成

本报讯(记者 赵洁)近日，记者从卫生部获悉，截至今年11月底，我国农村地区已建设783万户无害化卫生厕所，超额完成2009、2010年度项目任务，为实现“十一五”规划全国农村卫生厕所普及率达到65%的目标提供了有力保障。

国家重大公共卫生服务项目实施以来，各级地方政府加大了对农村改厕工作的重视程度，多数省份将农村改厕项目列为为民办实事工程，层层分解任务，落实责任。各级项目执行部门认真总结经验，加强规范项目管理，大胆创新工作模式，超额提前完成任务。

据介绍，2009年度、2010年度农村改厕项目共投人中英财政资金28.65亿元，在全国30个省（区、市)和新疆生产建设兵团支持建设758万户农村无害化卫生厕所。各地卫生部门积极争取地方财政的支持，通过中央项目的带动，地方各级共投人改厕配套资金23.04亿元，其中省级项目配套资金14.55亿元。

“在推进项目进程中，一些省份起到了示范带头作用。如江苏省委、省政府将农村改厕作为重大民生工程加以推进，专门制定管理办法，各级配套资金足额落实，连续两年大幅度超额完成国家下达的任务。”卫生部新闻发言人邓海华说，“西藏、浙江、辽宁、北京、天津等省、区、市也都超额或提前完成了项目任务。”

# 农机化发展实现历史性跨越

(上接第一版）“十一五"期间补贴资金规模逐年大幅增加，受益地区和受益农户不断扩大，中央财政累计安排农机购置补贴资金361亿元，带动地方和农民投人约1105亿元，补贴购置农机具1078万台（套），受益农户达到906万户。拉动农机工业快速发展，规模以上农机企业产值年均增长超过20%。“十一五"农业机械保有量大幅度增长，农机装备结构不断优化，力度之大前所未有。预计2010年，全国农机总动力达到9.2亿千瓦，比2005年增长34%。

农机化水平显著提高，农业生产方式实现了从人畜方向机械化作为主的历史性跨越。预计2010年，全国农作物耕种收综合机械化水平达到52%，比2005年提高16个百分点，农业生产进人机械化作为主的新时代。小麦基本实现了生产全过程机械化，水稻、玉米等主要农作物生产机械化快速推进。薄弱环节机械化取得重大突破，水稻种植耕种收水平从2005年的7.1%提高到今年的20%以上，水稻机收水平从33.5%提高到60%以上，玉米机收水平从44%提高到25%。马铃薯、油菜、棉花、花生等机械化从无起步，机械化作业面积均有较大增长。机械化保护耕作、深松整地、精量播种、化肥深施、秸秆还田、高效植保等具有良好生态效益和社会效益的农机化技术得到进一步推广，应用范围不断扩大。

农机合作社等服务组织蓬勃发展，农业社会化服务能力不断增强。预计2010年，全国农业机械化作业服务组织达到18.5万个，作业服务总收人达到3700亿元，比2005年增长63%。特别是农机专业合作社从无到有，2010年超过2万个，人社人数达到3.3万人。以农机专业合作社组织为龙头，农机大户为主体，农机户为基础，农机中介组织为纽带的农机社会化服务体系初具规模，服务模式不断创新，作业环节正从产中向产前、产后扩展，作业领域从种植业发展到畜牧业、农产品加工等领域。农机作业服务成为农业社会化服务发展的突出亮点和农民持续增收的重要渠道。

求木之长，必固其本。发射一营的“正规化”，不仅仅体现在战备、训练、工作、生活这些“硬环境”上，更体现在风清气顺、官兵和谐这些“软环境”上。

考学提干、立功受奖、选改士官，这些涉及官兵切身利益的“敏感问题”，是检验基层组织的一面“镜子”。一营用“规章制度"把门，让“素质能力”说话。那年，上级破格分给一营2名优秀士兵预提军官人选，可所属两个连队却报上来3人。对照上级条件，3人都“够格”。面对“三进二"难题，营党委不对上推透，不对下指责。党委会上，拿出历年综合素质考核结果全面衡量，立马“划"出前两名。

一连战士卢全文，被排在预提人选第一位，直到填表时，才知道自己“榜上有名”。营里几位老兵赞扬说，近10年，全营共有18名战士提干，39人立功，近200人选改士官，人人都服气，上下都满意。

发射一营的正气树得牢，靠的是“一班人"在全营官兵中坚持敬业奉献的模范、身体力行的模范、勤学上进的模范、公道正派的模范。

在一营见到这样一些细节：每次大小集合站队，都是营连主官站在队伍最前面；卫生区内都有营连领导的姓名牌；菜地里都有营连领导的责任田；组织专业体能考核，“一班人"与官兵一同过关；开展战法训法研修，与官兵逐个课题练；合成训练，坚持靠前指挥，一线组训。

近10年来，营党委历届班子成员有6人考上硕士研究生，在军内外刊物发表论文20多篇，取得科研革新成果17项。由“一班人"探索总结的串讲训练法被二炮列人十大训法并在部队推广，所属两个连队长都是基地的标兵连队，其中发射一连党支部被中组部树为“全国基层建设先进单位”。

# 我的营队我的家

从发射一营走出去的官兵，常常对山凹里那些营区的点点滴滴魂牵梦萦。

2009年“八一"前夕，营党委对曾在一营工作过的战友发出一封特殊的慰问信，介绍营队全面建设取得的新成果。很快，思念、问候、建言、对策、感激等多种情愫伴着书信、短信、邮件、电话等方式像雪片一样飞回营区，数百种反馈信息中，都表达着一个共同的感受：我的营队我的家

今年年初，营里来了一位特殊的客人。她是一连士官张怀生前爱的  
人，这次是替爱人回营探亲来了！

漫步营区，山涧小道曲折幽径，亭台楼阁点缀山峰，湖中鱼蟹相映成趣，道边随处可见官兵创作的雕塑作品，像微虫长空、鹤舞九天、鹿鸣深山等，惟妙惟肖，精美别致。途经忠诚墙、砺剑亭、精武路、兵法林，她静静地坐下来，听战士们争相为她讲起营里的点点滴滴.··

“嫂子，你看那两棵枝繁叶茂的桂花树吧。它们的‘籍贯'是湖北的，那是一级士官易鹏几年前利用休假时机，不远千里从湖北老家带回后栽下的。每年大伙儿探亲休假时从那时都要带上家乡的花籽、菜种、树苗等种到营区的，有的还从家乡带来了特别好使的菜刀和锄头。”

“东边的“世纪园"和“士兵文化园"还是我们的主意呢！为了业余时间娱乐有去处，营党委听了我们的建议后，全营官兵齐上阵，找来推车、塑铁、铁锹等工具，开山挖坟运半年，硬是开辟出了一块3500多平方米的多功能娱乐场，现在深受官兵喜爱”“我们一点也不孤单！在家是父母的焦点，在这里依然是焦点。每年干部都有一本‘知兵录'和‘我们的'成长日记”。上面记录着大家的家庭情况、文化程度、性格特点、爱好特长、成长进程、思想变化、工作情况等；营干部每天都要做到“四个转一转”：早晨转一转，看有无病号；吃饭转一转，看大家吃得香不香；业余时间转一转，看战士们在干什么；媳妇后转一转，看我们睡得好不好。

“这就是我们的家，爱这个家不仅成了传统，更内化为大家的生活习惯了！”

一路走来，她终于明白了爱人在什么也在血雨化行期间还要扶助回到连队参加《操作规程》修编，为什么留给她的遗言竟是“这个山沟没装好，有时间多替我回来看一看”

# 该干啥，如何干，大家心知肚明

作为一个老典型，一营一年四季总是“贵客盈门”一一有总结经验的、有参观见学的、有检查指导的、有考核评比的，最多时，一天接待两三拨“工作组”。关心一营的人最担心的是，一营的正常秩序会不会被“迎来送往”打乱？熟知一营的人最放心的，是一营永远都有自己的“行为准则”一一不怕工作忙，不怕任务多，事事讲规矩，处处有准绳。

2005年5月，一营讲定期受训人任务：一项是参加第二炮兵首届军事技术大比武，一项是接受基地实战操作训练考核。两班顶住、双重考验，既检验他们的综合素质，又检验他们的统筹功力。一营采取“组合抽训”、交替推进”的训练方法，有效化解了“节拍冲突、任务交叉"的矛盾，比武站上我第一关，考核榜上得优秀。

既给得起“自家”考，更经得起“大家”考。去年秋天，发射一营正在执行一项任务，旅里临时赋予他们“一个‘应急课目”——迎接兄弟单位参观见学。来自基地数千名带队者又是抓基层的“行家里手”，他们深入营连，像“过筛子”一样把营里的工作“摸”了个遍，给出这样的评价，这个营忙而有序，章法分明，善于统筹，全面过硬，堪称基层建设科学发展的典范。

在发射一营，党委支部工作有谱，干部战士言行有教，这个“谱"就是基层建设纲要，这个“数"就是法规制度。该干啥，如何干，大家心知肚明，都揣着一本明白账。旅政治部主任张炳介绍，新的基层建设纲要出台后，营党委及时对照新标准新要求，不断校正基层建设的“谁心”，理清发展思路，不分心走神，集中精力谋发展。
