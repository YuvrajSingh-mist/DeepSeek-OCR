### 4. 数词的使用法： 

1) 表示年份：2002: twenty thousand and two; 97b: nineteen seventy-six. 

2) 表示日期：12月1日：Dec. 1st 或 the first of December; 

2002年11月8日：Nov. 8th, 2002. 

3) 表示时刻：5:15 → five fifteen 或 a quarter past five;
8:30 → eight thirty 或 half past eight;
10:45 → ten forty-five 或 a quarter to eleven. 

4) 表示偏号：Room 105 → Room one O five; Bus No.13 → Bus Number Thirteen;
P.5 → Page Five; Tel. No. 7658659 → Telephone Number seven-six-five-eight-six-five-nine. 

5) 小数的读法：5.7 → five point seven; 0.16 → zero point one six. 

6) “半”的表示：1/2 → half; 半小时 → half an hour; 1.5小时 → one and a half hours 或 one hour and a half. 

7) 序数词前面加the时，表示顺序，如a/an时表示“再一、又一”。如：The third lesson is rather difficult. (第三课相当难) / Should we read the text a third time? (我们把课文读第三遍，好吗?)
